link 和 @import 的区别？

  1. link是HTML外部资源链接标签，不仅可以用来加载CSS，还可以用来创建站点图标、定义RSS喔！
    @import 是CSS的规则，只能引入css文件。
  2. link加载资源，是在页面载入的同时进行的；@import 则需要页面完全载入之后再进行CSS文件的加载，而且加载顺序不一定符合预期（容易出现无样式页面闪烁一下再恢复正常的情况）。
  3. link 若作为XHTML标签需要被关闭：<link />
  4. 可以通过JS控制DOM，插入 link 标签来改变样式，@import不行喔。

CSS样式引入优先级？

理论上呢，行内>嵌入>链接>导入。
但实际上由于嵌入、链接、导入方式均在文件头部，类似就近原则，谁离相应代码最近，就显示谁的样式（后来者居上，把前浪拍死在沙滩上覆盖掉啦！）
